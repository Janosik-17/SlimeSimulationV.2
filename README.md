# SlimeSimulationV.2

Currently has two branches, the program on the main branch is the orignal simulation with primiive ants that only follow one type of pheromone, on the test branch there is a version with a complete ant anglorythm in which the agents are inclined to find food and return home. To get nice results try using these settings on the test branch. 



<code>{
  "DecayRate": 0.89,
  "SmellDistance": 12,
  "SmellAngle": 2,
  "SlimeSpeed": 1,
  "TurningSpeed": 0.4,
  "WigglyPathCoeff": 0.26,
  "DepositPheromoneAmount": 20,
  "FoodEmissionStrength": 100,
  "FoodNutrition": 200,
  "FoodIsForever": false
}
</code>
